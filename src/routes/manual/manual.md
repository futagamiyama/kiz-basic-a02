# BASIC Step Debugger — Manual

シンプルな BASIC 風言語のステップ実行デバッガです。1 行ずつ実行しながら変数・配列・呼び出しスタックの変化を観察できます。

## 操作ボタン

| ボタン | ショートカット | 動作 |
|---|---|---|
| **1step** | `Ctrl+1` | 1 行ずつ実行。空白行・コメント行・ラベルのみの行は飛ばします |
| **run** | `Ctrl+2` または `Enter` | 残りを最後まで実行。`goto`（無条件）を含むプログラムはエラー扱い（無限ループ防止） |
| **return** | — | 変数・配列・関数・呼び出しスタック・キャンバスをリセット（コードは保持） |
| **clear** | — | エディタの内容を消去 |

`Ctrl+1` / `Ctrl+2` はエディタ内を含む**ページ内のどこでも**有効です。`Enter` は背景・キャンバスペインなど、エディタ・ボタン・入力フィールド以外にフォーカスがあるときに run として動作します。

> 注：Windows / Linux のブラウザでは `Ctrl+1〜9` がタブ切替と競合します。ページ内では `preventDefault` でブロックしていますが、ブラウザによっては動作が異なる場合があります。macOS ではタブ切替が `Cmd+1〜9` なので競合しません。

## 基本文

### 代入

```
a = 10
s = a + 5
name = "hello"
```

### print（略記 `p`）

式を評価して Output に表示します。`p` を略記として使えます（後ろに空白＋式が続くとき、または単独）。`p=5` は変数 `p` への代入として扱われ、衝突しません。

```
print 1 + 2
p sin(pi / 2)
p "x = " + x
```

### end

プログラムを終了します。

### コメント

`#` または `'` で始まる行はコメント。

## ラベルと制御文

### ラベル

行の先頭に `name:` を書いて目印にします。`name:` だけの行も、`name: 文` の形（ラベル + その行に文を続ける）も同じ意味です。

```
loop1:
loop1: s = s + n
```

### goto

無条件ジャンプ。**run モードでは無限ループ防止のためエラーになります**（`1step` でのみ実行可）。

```
goto loop1
```

### if … goto

条件付きジャンプ。条件が真ならジャンプ、偽なら次の行へ。`run` モードでも実行可能です（終了し得るため）。

比較演算子: `<`、`<=`、`>`、`>=`、`=`、`<>`

```
if n <= 10 goto loop1
```

### gosub / return

サブルーチン呼び出し。`return` で `gosub` の次の行に戻ります。ネスト可、再帰可。

```
gosub mysub
end
mysub:
print 1
return
```

## 演算子と優先順位

優先順位（低 → 高）:

1. `+`、`-`（加減）
2. `*`、`/`（乗除）
3. 単項 `-`、`+`
4. `^`（冪、**右結合**）
5. リテラル・変数・関数呼び出し・括弧

例:

- `-2^2` → `-4` （`-(2^2)`）
- `2^3^2` → `512` （`2^(3^2)`）
- `2^-3` → `0.125`

## 定数

| 名前 | 内部値 | 表示 |
|---|---|---|
| `e` | `Math.E` | 2.71828 |
| `pi` | 3.14159 | 3.14159 |

代入は不可。`e = 3` はエラー。

## 組込関数

| 関数 | 説明 |
|---|---|
| `sin(x)`, `cos(x)`, `tan(x)` | 三角関数（ラジアン） |
| `sqrt(x)` | 平方根 |
| `ln(x)` | 自然対数 |
| `log(x)` | 常用対数 log₁₀(x) |
| `log(base, x)` | 任意の底の対数 |

## 配列

```
dim a[10]
a[0] = 1
a[i] = a[i-1] * 2
print a[5]
```

- 0-based、サイズは `dim` で決定（式 OK：`dim a[n+1]`）
- 添字は整数式。範囲外はエラー
- 入れ子可: `a[b[i]]`

## ユーザ関数定義（def）

単一の式からなる関数を定義します。

```
def f(x) = x*x + 1
def dist(a, b) = sqrt(a*a + b*b)
print f(3)
print dist(3, 4)
```

- 引数は呼び出し時にだけバインドされ、グローバル変数を変更しません
- 別のユーザ関数を本体から呼べる（**定義順**に注意）
- 再帰可（JS スタック上限まで）
- 本体は式 1 本のみ。式中に比較演算子は書けないため、条件分岐を含む関数は不可

## グラフィックス

描画文を含むプログラムを実行すると、**別ウインドウ**（400 × 400 のキャンバス）が自動で開きます。

| 文 | 動作 |
|---|---|
| `cls` | キャンバスをクリア |
| `plot x, y` | 点を描画 |
| `line x1, y1, x2, y2` | 線分を描画 |
| `color r, g, b` | 描画色（各 0–255） |
| `window xmin, ymin, xmax, ymax` | 論理座標系を設定。Y 軸は上向き |

`window` 未設定時はピクセル座標（左上原点、Y 下向き、400 × 400）。

例: sin カーブ

```
window -1, -1.2, 7, 1.2
color 180, 180, 180
line -1, 0, 7, 0
line 0, -1.2, 0, 1.2
color 200, 0, 0
x = -1
loop1:
plot x, sin(x)
x = x + 0.05
if x < 7 goto loop1
end
```

## デバッグパネル

右側に表示されます:

- **Constants**: 定数 `e`、`pi`
- **Debug**: Current Line、Status、エラー
- **Functions**: 定義済みユーザ関数の一覧
- **Variables**: スカラ変数の値
- **Arrays**: 配列名と要素
- **Call Stack**: `gosub` の戻り先（`return → line N`）
- **Output**: `print` の出力

## 命名規則

- 識別子は英字または `_` で始まり、英数字・`_` が続きます
- **大文字小文字は区別しません**（内部的にすべて小文字として扱われます）
- `print`、`goto`、`gosub`、`return`、`if`、`end`、`dim`、`def`、`cls`、`plot`、`line`、`color`、`window` は予約語
- 組込関数名・定数名は変数・配列・ユーザ関数の名前として使えません

## サンプル

### 1 から 10 の和

```
s = 0
n = 1
loop1: s = s + n
n = n + 1
if n <= 10 goto loop1
print s
end
```

### 階乗（gosub で）

```
n = 5
result = 1
i = 1
loop1: result = result * i
i = i + 1
if i <= n goto loop1
print result
end
```

### 関数とグラフを組み合わせ

```
def f(x) = sin(x) * cos(2*x)
window -3, -1.2, 3, 1.2
color 100, 0, 200
x = -3
loop1:
plot x, f(x)
x = x + 0.02
if x < 3 goto loop1
end
```
